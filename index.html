Welcome to the NovOS core repo!

This is a basic page for anyone who opens this repo in a web browser.

For people who already have NovOS installed: Run sudo pacman -Syu in your terminal and update all your packages.

For people don't have NovOS: Download the offical ISO from our website and install it.

 

NovOS is based on Arch linux and was made from the Ezarcher project from eznix.
